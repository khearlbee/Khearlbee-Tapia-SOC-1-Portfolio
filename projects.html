PROJECT 1: SIEM Deployment & Log Correlation

Week 1-2

Deployed Splunk Enterprise and successfully ingested the BOTSv2 dataset containing 68+ million security events. Built operational dashboards and developed SPL queries to identify threat indicators and attack patterns.

🎯 Objective

  • Set up production-grade SIEM environment
  • Ingest and normalize large-scale security data
  • Create baseline searches and detection queries
  • Build dashboards for SOC monitoring

🛠️ Tools & Technologies

Splunk Enterprise SPL (Search Processing Language) BOTSv2 Dataset Dashboard Design

📊 Key Findings

  • Processed 68M+ events from August 2020
  • Identified top 10 source IPs with malicious activity
  • Detected HTTP anomalies and suspicious ports
  • Created 6+ operational dashboard panels

📝 Deliverables

  • Splunk SIEM configuration documentation
  • 50+ SPL queries with descriptions
  • Dashboard screenshots and panel specifications
  • Performance tuning notes and optimization tips

PROJECT 2: Threat Detection & Alert Triage

Week 2-3

Developed comprehensive threat hunting methodology and created detection queries for identifying indicators of compromise. Implemented alert triage framework and severity assessment procedures following industry best practices.

🎯 Objective

  • Hunt for attack signatures and anomalous behavior
  • Write detection queries for known attack patterns
  • Implement alert triage methodology
  • Develop false positive analysis framework

🛠️ Tools & Technologies

Splunk Detection MITRE ATT&CK Alert Tuning Severity Assessment

🚨 Detection Queries Developed

  • Brute force attack detection (EventCode 4625)
  • Suspicious process execution (PowerShell, cmd.exe)
  • Unusual port connections (non-standard ports)
  • Data exfiltration attempts (large outbound transfers)
  • DNS tunneling detection (anomalous query patterns)
  • Lateral movement (administrative port scanning)

📝 Deliverables

  • 6+ documented detection queries
  • Alert triage matrix and severity/priority framework
  • False positive analysis with tuning recommendations
  • 3+ alert runbook templates
  • Detection metrics and KPI tracking
