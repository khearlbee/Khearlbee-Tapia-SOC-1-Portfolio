About This Portfolio

Khearl Bee Tapia

I am Khearl Bee Tapia, a cybersecurity student with a growing focus on security operations and incident response. My interest in this field comes from wanting to understand not just how systems get attacked, but how analysts detect, investigate, and respond to those threats in real time.

This portfolio reflects my hands-on journey through SOC fundamentals — from deploying SIEM platforms and writing detection queries, to analyzing network traffic at the packet level and documenting full incident response procedures. Each project pushed me to think like both an attacker and a defender, and to translate technical findings into clear, actionable reporting.

I'm continuing to build my skills toward a career as a SOC Analyst, and I welcome any feedback, questions, or opportunities to connect.

Purpose

This portfolio documents my hands-on learning in security operations and incident response through a comprehensive 4-week lab project for the Cyber Threat Monitoring class. Each project represents a critical component of SOC operations, demonstrating practical skills in SIEM deployment, threat detection, network forensics, and incident response procedures.

Learning Objectives

  • Deploy and configure production-grade SIEM platforms (Splunk Enterprise)
  • Write effective detection queries using SPL and other query languages
  • Perform threat hunting and identify indicators of compromise
  • Analyze network traffic at the packet level using Wireshark
  • Develop professional incident response procedures and documentation
  • Apply industry frameworks (NIST, MITRE ATT&CK) to security operations
